# 🧑‍💻 Abdelrhman Beseiso - Personal Portfolio

> A personal portfolio website built using Django to showcase my work, skills, and allow visitors to contact me securely.

## 📝 Description

This is the source code for my personal portfolio website. The site is built using **Django** and hosted on platforms like **Render**, **Heroku**, or **Vercel**, and includes a **secure contact form** that allows visitors to send me messages directly from the site.

The contact form sends emails through a secure backend service, ensuring user privacy and preventing email harvesting by bots.

---

## 🛠 Features

- Clean and responsive design.
- Section for About Me, Projects, and Skills.
- Contact form with server-side validation.
- Secure email sending using environment variables.
- Easy to customize and extend.

---

## 🚀 Technologies Used

- **Python**
- **Django**
- HTML5 / CSS3 / JavaScript (Vanilla)
- resend service https://resend.com (for sending emails securely)
- Environment Variables for sensitive data
- Optional: TailwindCSS or Bootstrap for styling
